Start typing right away. The text that will appear on this page will be with a cryptic script, and it will be very hard to read. If you would like to change the visuals of it you can use the controllers to do so. Typing in cryptic - almost unreadable script can be very liberating. Try it out for a while. If you would like to translate your writings into something readable when you are done there are a few ways to do so.

The text will go up and down all over the page. If you are ending up outside the screen use the key "=" to navigate up again and "+" to jump down.

Cryptomenysis Patefacta is a book written by John Falconer in 1685. In the book, Falconer explains a number of different code languages that can be used in script, speech and with hand gestures.

On this website, you can use a language based on a Bi-literary Alphabet. It is modified so that all letters get a direction based on their phonetic groupings in the alphabet. All the letters that are pronounced similarly are given the same direction.
